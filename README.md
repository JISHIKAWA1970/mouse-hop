# Mouse Hop

Mouse Hop は、マルチディスプレイ環境でマウスカーソルを次のディスプレイ中央へ移動する Windows 向けの軽量常駐アプリです。

## v0.3 の機能

- 起動するとタスクトレイに常駐します。
- 初期ホットキーは `Ctrl + Alt + F12` です。
- タスクトレイの `設定` からホットキーと移動方式を変更できます。
- 設定したホットキーを押すたびに、現在カーソルがあるディスプレイを判定し、選択した移動方式に従って次のディスプレイ中央へ移動します。
- `ループ` では最後のディスプレイの次に最初のディスプレイへ戻り、`往復` では端のディスプレイで移動方向を反転します。
- 現在カーソルがあるディスプレイを判定できない場合は、内部の巡回カウンターで次のディスプレイへ移動します。
- タスクトレイの右クリックメニューから以下を実行できます。
  - `次のディスプレイへ移動`
  - `設定`
  - `一時停止` / `再開`
  - `終了`
- 終了時にグローバルホットキーを解除します。
- ホットキー登録に失敗してもアプリは終了せず、タスクトレイに常駐し続けます。

## 対応環境

- Windows 10
- Windows 11
- .NET 8 SDK

## ビルド方法

```powershell
dotnet build
```

Release ビルドを作成する場合は以下を実行します。

```powershell
dotnet build -c Release
```

## 実行方法

```powershell
dotnet run --project src/MouseHop/MouseHop.csproj
```

起動後、通知領域に `Mouse Hop` のアイコンが表示されます。

## 使い方

1. Mouse Hop を起動します。
2. 初期設定では `Ctrl + Alt + F12` を押します。
3. カーソルが現在のディスプレイの次のディスプレイ中央へ移動します。
4. もう一度同じホットキーを押すと、さらに次のディスプレイ中央へ移動します。
5. タスクトレイアイコンを右クリックし、`終了` を選ぶとアプリを終了します。

## マウス追加ボタンで使う場合

Mouse Hop は、ユーザーがキーボードで複雑なショートカットを押すためのアプリではありません。マウス設定アプリ側で追加ボタンにショートカットを割り当て、そのショートカットを Mouse Hop が受け取る設計です。

初期設定では、マウス設定アプリ側で追加ボタンに `Ctrl + Alt + F12` を割り当てると、Mouse Hop をマウスボタン 1 つで実行できます。

Logi Options+ などのマウス設定アプリで別のキー組み合わせを割り当てたい場合は、Mouse Hop のタスクトレイメニューから `設定` を開き、同じキー組み合わせをホットキーとして登録してください。`Ctrl` / `Alt` / `Shift` / `Win` などの修飾キー付きショートカットに対応しています。`F13` のような単独キーも、入力できる環境では登録できます。

## 設定

1. タスクトレイアイコンを右クリックします。
2. `設定` を選びます。
3. 現在のホットキーと移動方式を確認します。
4. ホットキーを変える場合は `変更` を押します。
5. 次に押したキー組み合わせが新しいホットキーとして保存されます。
6. 移動方式を変える場合は、`移動方式` から `ループ` または `往復` を選びます。

設定は AppData 配下の `MouseHop/settings.json` に保存され、次回起動時にも反映されます。既存の settings.json に移動方式が存在しない場合は、初期値として `ループ` を使用します。ホットキー変更時は古いホットキーを解除してから新しいホットキーを登録します。

## 移動方式

接続中のディスプレイは、画面上の配置に基づいて上から下、同じ高さでは左から右の順に並びます。メインディスプレイが左端または上端にない構成や、負の座標を含む配置にも対応します。

### ループ

現在までの挙動と同じ方式です。最後のディスプレイまで移動したら、次は最初のディスプレイへ戻ります。

例: `1 → 2 → 3 → 1 → 2 → 3`

### 往復

端のディスプレイまで移動したら、前回の移動方向を反転して戻ります。

例: `1 → 2 → 3 → 2 → 1 → 2 → 3`

2 画面の場合は、ループも往復も実質的に `1 → 2 → 1 → 2` です。

## 既知の制限

- インストーラーはありません。
- Windows 起動時の自動起動登録はありません。
- 移動時のオーバーレイ表示はありません。
- 設定したホットキーが他のアプリで使用中の場合、ホットキー登録に失敗することがあります。その場合もタスクトレイには常駐し、右クリックメニューの `次のディスプレイへ移動` は使用できます。
