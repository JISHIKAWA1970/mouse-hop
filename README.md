# Mouse Hop

Mouse Hop は、マルチディスプレイ環境でマウスカーソルを次のディスプレイ中央へ移動する Windows 向けの軽量常駐アプリです。

## v0.3 の機能

- 起動するとタスクトレイに常駐します。
- 初期ホットキーは `Ctrl + Alt + F12` です。
- タスクトレイの `設定` からホットキーと移動方式を変更できます。
- 設定したホットキーを押すたびに、現在カーソルがあるディスプレイを判定し、選択した移動方式に従って次のディスプレイ中央へ移動します。
- `ループ` では最後のディスプレイの次に最初のディスプレイへ戻り、`往復` では端のディスプレイで移動方向を反転します。
- 現在カーソルがあるディスプレイを判定できない場合は、内部の巡回カウンターで次のディスプレイへ移動します。
- タスクトレイの右クリックメニューから以下を実行できます。
  - `次のディスプレイへ移動`
  - `設定`
  - `一時停止` / `再開`
  - `終了`
- 終了時にグローバルホットキーを解除します。
- ホットキー登録に失敗してもアプリは終了せず、タスクトレイに常駐し続けます。

## 対応環境

- Windows 10
- Windows 11
- .NET 8 SDK
