# Mouse Hop

Mouse Hop は、マルチディスプレイ環境でマウスカーソルを次のディスプレイ中央へ移動する Windows 向けの軽量常駐アプリです。

## v0.1 の機能

- 起動するとタスクトレイに常駐します。
- `Ctrl + Alt + M` を押すたびに、現在カーソルがあるディスプレイの次のディスプレイ中央へ移動します。
- 最後のディスプレイの次は最初のディスプレイへ戻ります。
- 現在カーソルがあるディスプレイを判定できない場合は、内部の巡回カウンターで次のディスプレイへ移動します。
- タスクトレイの右クリックメニューから以下を実行できます。
  - `次のディスプレイへ移動`
  - `一時停止` / `再開`
  - `終了`
- 終了時にグローバルホットキーを解除します。

## 対応環境

- Windows 10
- Windows 11
- .NET 8 SDK

## ビルド方法

```powershell
dotnet build
```

Release ビルドを作成する場合は以下を実行します。

```powershell
dotnet build -c Release
```

## 実行方法

```powershell
dotnet run --project src/MouseHop/MouseHop.csproj
```

起動後、通知領域に `Mouse Hop` のアイコンが表示されます。

## 使い方

1. Mouse Hop を起動します。
2. `Ctrl + Alt + M` を押します。
3. カーソルが現在のディスプレイの次のディスプレイ中央へ移動します。
4. もう一度 `Ctrl + Alt + M` を押すと、さらに次のディスプレイ中央へ移動します。
5. タスクトレイアイコンを右クリックし、`終了` を選ぶとアプリを終了します。

## ディスプレイの巡回順

v0.1 では、接続中のディスプレイを画面上の配置に基づいて上から下、同じ高さでは左から右の順に巡回します。メインディスプレイが左端または上端にない構成や、負の座標を含む配置にも対応します。

## 既知の制限

- 設定画面はありません。
- インストーラーはありません。
- Windows 起動時の自動起動登録はありません。
- 移動時のオーバーレイ表示はありません。
- `Ctrl + Alt + M` が他のアプリで使用中の場合、ホットキー登録に失敗して起動できないことがあります。
